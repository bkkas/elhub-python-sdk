# Usage

To use ElHub Python SDK in a project

```
import elhub_sdk
```

## Requirements

The package needs the following environment variables:

**Key and cert files for signing to Elhub**
- KEY_FILE
- CERT_FILE


**WSDL files**
WSDL files will point against SoapUI Mock services or ElHub environment.
- WSDL_QUERY
- WSDL_POOL_METERING
